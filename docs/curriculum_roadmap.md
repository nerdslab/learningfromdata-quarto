### Key findings from the current notebooks

Two facts reshaped this from "write a lot of new content" into "mostly migration":

1. **No Bayes content exists** in 3a or 3b. "Bayes deprioritized" is a no-op —
   nothing to remove.
2. **The CLT/LLN capstone already exists — in 3b.** The "Sampling Distributions:
   Where t-tests Come From" section (`Notebook_3b:467`) *is* a CLT/LLN demo:
   repeated samples → distribution of the mean → it narrows → `SE = σ/√n`.
   Already written, tested, with embedded outputs. Migrate it into 3a rather than
   author from scratch.

### Target: 3a → **Probability** (L8) ✅

Narrative spine: build distributions → expectation/variance → LLN (averages
converge) → CLT (averages go Gaussian, `SE = σ/√n`).

- **Keep:** What is probability; Random variables; Expectation & variance;
  Coin-flip simulation (already Bernoulli/Binomial *by generating process* — the seed).
- **New (the only net-new writing):** the **distributions zoo** — Uniform,
  Poisson, Exponential, Gaussian, each taught *by its generating process*, with
  E/Var formalized per distribution.
- **Migrate in from 3b:** "Sampling Distributions" + "SE shrinks with √n" →
  reframed as the **LLN → CLT capstone**.
- **Move out:** all covariance/correlation (see split below).

### Target: 3b → **Statistics 1** (L9) ✅

- **Loses** the sampling-distribution material to 3a (~95 lines freed).
- **Gains** the *statistical* covariance/correlation from 3a.
- **Keeps:** framing a question; test statistics & p-values; confidence intervals;
  t-test deep dive; two-sample t-test types; ANOVA; paired t-test.
- **Opens** by cashing in 3a's CLT: "because the sample mean is ~Gaussian, the
  t-statistic has a known distribution" — clean handoff.
- **Type I/II introduced here** (α = Type I error; power = 1−β). The existing α
  and "fail-to-reject ≠ accept" callouts already imply it — this is a labeling
  pass, not new content.

### Target: 3c → **Statistics 2** (L10) ✅

- **Keeps:** bootstrap; permutation tests.
- **Type I/II quantified** — the "Noise, Sample Size, and Power" demo needed for
  this already exists at `Notebook_3c:347`. Reframing, not writing.

### Covariance split ✅ (resolves Open Decision: "covariance's home")

The 3a covariance content is mostly *geometric* (shape-of-a-data-cloud, tilted
clouds, interactive shape explorer) — that's PCA prep, not t-test prep. Split it:

- **Statistical** part (correlation coefficient, correlation matrix, correlated-
  variables sim, correlation in a real dataset) → **L9 Statistics (3b).**
- **Geometric** part (covariance as shape of a data cloud, tilted cloud, three-
  shapes comparison, interactive shape explorer) → **earmark for M5 PCA prep.**
  Re-invoke the covariance matrix there: "its eigenvectors are the principal
  components." Same spiral pattern as Type I/II.
